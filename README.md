# D-ask Frontend

대덕소프트웨어마이스터고등학교 학생을 위한 AI 기반 학교 정보 QnA 및 학사 일정 서비스 **D-ask**의 프론트엔드입니다.




---

## 소개

분산되어 있는 학교 문서를 통합하여, 학생들이 자연어로 학교 정보를 쉽게 찾을 수 있도록 돕습니다.

**주요 대상**: 재학생 및 신입생



## 주요 기능

### 메인 페이지
- 서비스 소개 및 페이지 네비게이션
- 사용 가이드와 FAQ

### QnA 페이지
- 채팅 기반 AI 질의응답
- 실시간 응답 표시

### 캘린더 페이지
- 월별 학사 일정 조회
- 주요 행사 시각화



## 기술 스택

순수 JavaScript로 구현

- HTML5, CSS3, Vanilla JavaScript
- Fetch API를 통한 백엔드 연동
- Figma를 사용한 UI/UX 디자인

**외부 프레임워크 및 라이브러리 미사용**



## 디자인 특징

- 직관적인 네비게이션
- 친근한 인터페이스로 AI 서비스 접근성 향상
- 하늘색·남색 계열의 통일된 컬러 시스템


### 향후 개선 사항

-로그인 및 사용자 인증 UI 추가

-반응형 디자인 고도화

-접근성(ARIA) 개선

-실제 배포 환경 대응 UI 수정
